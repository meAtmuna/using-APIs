# Homepage with APIs

THis is a custom homepage project built using HTML, CSS, and JavaScript. It shows useful daily information with a clean UI.

## Features

* NASA Astronomy Picture Of the Day (APOD)
* Live Time (updates every Second)
* Current Weather (based on user location)
* Quote of the Day
* Current Date

## APIs Used

* NASA APOD API -> background image
* Open-Mateo API -> weather
* DummyJSON API -> quotes

## How To Use 

1. Download or clone this project 
2. Open the folder
3. Run index.html in your browser

Allow location permission to see weather
